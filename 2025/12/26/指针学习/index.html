一、什么是指针

1.1 基本概念

指针是内存地址的抽象，是内存中的一个地址编号
以电脑中的创建文件夹为例：

  • 变量 = 文件本身（存储内容）
  • 指针 = 文件索引（指向位置）
  • 内存 = 整个硬盘（存储空间）

1.2 指针的声明与初始化

声明方式： 指针指向的数据的数据类型 * 指针变量名;
例：
int* p;
char* p;
void* p;//万能指针，可以指向任意类型
初始化： 指针变量=&普通变量
例：
p = &doc;
二者可以合并同时进行
例：
int doc = 10;
int *p = &doc;
